# StockDoge ($SDOGE)

Official landing page for **StockDoge** — the meme token on Robinhood Chain, fair-launched via [Pons Launchpad](https://pons.family/launchpad).

## Open the site

This is a static site. No build step required.

- Double-click `index.html`, or
- Serve the folder with any static server, e.g. `npx serve .`

## Structure

```
index.html          # page markup
css/style.css       # theme, layout, animations
js/main.js          # candles, reveals, counters
assets/logo.png     # StockDoge logo
assets/banner.png   # hero / about banner
```
